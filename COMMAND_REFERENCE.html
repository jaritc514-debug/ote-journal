01 Startup Routine
Step 1 — Launch TradingView (PowerShell)
powershell 
$tvPath = (Get-AppxPackage -Name "*TradingView*").InstallLocation
Start-Process "$tvPath\TradingView.exe" --remote-debugging-port=9222
Step 2 — Verify Connection (PowerShell)
powershell 
Invoke-WebRequest -Uri 'http://localhost:9222/json' -UseBasicParsing
Step 3 — Launch Claude Code (CMD)
cmd 
cd %USERPROFILE%\tradingview-mcp && claude
02 Daily Commands
Quick Prefilter — single pair on chart
Checks Location + Direction + Action Matrix only. Fast go/no-go on current chart.
claude code 
run prefilter.txt
Batch Prefilter — all 20 FX pairs
Scans all pairs, returns ranked table. Outputs a ready-to-analyze command at the bottom — copy and paste it when ready.
claude code 
run batch-prefilter.txt
Full Analysis — single pair
Full OTE + HTF S&D + macro analysis on current chart. Outputs entry, SL, TP, grade, confluence, invalidation.
claude code 
run analysis-prompt.txt
Log a Trade — after full analysis
Generates pre-filled journal entry from analysis output. Copy each field into your journal. Fill manually: Position Size, Outcome, Emotions, Review after close.
claude code 
run log-trade.txt
03 Weekly Review
Step 1 — Export your trades
Go to your journal → Trade Log → click Export JSON → data copied to clipboard.
Step 2 — Run the review
Type the command, press Enter, then paste your JSON data and press Enter again.
claude code 
run weekly-review.txt

Startup

06 Rules Reminder
Equilibrium = NO TRADE — Hard filter, no exceptions. If price is in EQ, close the chart and move on.HARD RULE
Minimum R:R = 1:2 on all trades. Below this, no trade regardless of setup quality.
1% risk per trade — no exceptions.
Stop trading after 2 consecutive losses on the same day OR when FTMO daily drawdown limit is hit — whichever comes first.
Pre-trade plan must be written before entry. If you can't write it clearly in one sentence, don't take the trade.HARD RULE
